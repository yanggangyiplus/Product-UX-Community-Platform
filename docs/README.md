# 문서 구조

이 폴더는 프로젝트의 모든 문서를 포함합니다.

## 폴더 구조

```
docs/
├── requirements/      # 요구사항 정의서 (REQ-001 ~)
├── function-spec/    # 기능명세서 (FUNC-001 ~)
├── UX-research/      # 사용자 행동 분석 및 Pain Point 분석 + 개선안
├── IA/               # 정보구조도 (Information Architecture)
├── UX-flow/          # 유저 플로우 다이어그램
└── wireframes/       # Figma 와이어프레임
```

## 문서 작성 가이드

각 폴더의 문서는 다음 형식을 따릅니다:

- **요구사항 정의서**: REQ-001, REQ-002 형식으로 명명
- **기능명세서**: FUNC-001, FUNC-002 형식으로 명명
- **UX 리서치**: 사용자 행동 데이터 기반 분석 문서
- **IA**: 정보구조도 다이어그램 및 설명 문서
- **UX Flow**: 핵심 사용자 플로우 다이어그램
- **Wireframes**: Figma 링크 및 스크린샷
