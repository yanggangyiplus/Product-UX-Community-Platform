📌 Product-UX-Community-Platform
소셜 로그인 기반 커뮤니티 기획 + UX 설계 + 사용자 행동 기반 개선 프로젝트
신규 서비스 기획, UX 설계, 사용자 행동 기반 UX 개선까지 하나의 프로젝트에 통합한
기획·UX 포트폴리오 통합형 프로젝트입니다.

🎯 프로젝트 개요

이 프로젝트는 다음 3가지를 한 번에 보여주는 End-to-End 기획·UX 프로젝트입니다:

신규 서비스 기획
– 소셜 로그인 기반 커뮤니티 플랫폼 전체 구조 설계
– 요구사항 → 기능명세 → IA → UX → UI 까지 풀패키지 설계

UX 설계(User Flow 중심)
– 핵심 사용자 플로우 정의
– UX 예외 처리(지연·오류·Skeleton UI) 설계
– Figma 와이어프레임 제작

UX 개선(Research 기반)
– 사용자 행동 분석
– 핵심 Pain Point 도출
– 개선안 제시(Before → After 구조)
– IA 재정비 및 클릭 경로 최적화

✨ 프로젝트 목적
접근성 높은 커뮤니티 경험 제공
직관적인 탐색 UX 설계
운영자(Admin) 관리 비용 최소화
사용자 행동 기반 개선안까지 포함해 PO/기획/UX 역량을 완성형으로 표현

🔧 주요 구성
① 신규 서비스 기획 (소셜 로그인 커뮤니티)
기능	설명
소셜 로그인 (OAuth 2.0)	카카오/네이버/구글 인증

게시판	카테고리 기반 검색·정렬
게시글 CRUD	이미지 업로드·임시저장 포함
댓글/대댓글	기본 커뮤니티 기능
좋아요	반응 기반 랭킹
신고 기능	게시글·댓글 신고 및 중복방지
관리자 기능	신고 처리 / 회원관리 / 게시판관리

③ UX 개선(리서치 기반)
